How to install and Use Angular framework on Windows Computer

In this tutorial, We will be using Visual Studio to setup the framework for use.


1. To begin, download the 16.13.1 LTS version of nodejs here and install by running the executable.

2. Open up CMD and type "node â€“v" to verify installation. The output in the image below should be displayed inorder to verify installation.

3. Install Angular CLI by typing "npm install -g @angular/cli" into CMD.

4. Navigate to the directory you want to make your Angular app using the "cd pathname" command in CMD, where you replace "pathname" with your desired directory.

5. Run the command "ng new my-new-angular-app", where you can change "my-new-angular-app" to a name you desire.

6. Navigate to your new angular app directory by running the command "cd my-new-angular-app", where "my-new-angular-app" is the app name you chose.

7. Build app and start server with the following command "ng serve".

8. Go to http://localhost:4200/

9. Assuming you have installed VSC from previous tutorial, open VSC, launch, and select "Open a local folder".

10. Navigate to where you created your angular app and click "select folder" at the bottom right of the file explorer.

11. You now have the Angular framework installed where you can work on your Angular app through VSC.

12. To stop server, hit CTRL + C in CMD. If successful, you won't be able to connect to http://localhost:4200/


How to start using ASP.net on Windows Computer

In this tutorial, We will be using Visual Studio to setup the framework for use.


1. Download Visual Studio 2022 Community version here.

2. Execute the .exe file downloaded and follow the instructions to install VSC onto your system.

3. When prompted to select which workloads to install, select the options in the image below and click install at the bottom right.

4. Click the launch button after installation is completed and select the "Create a new project" option under "Get started".

5. Select the ASP.NET Web Application(.NET Framework) and click next.

6. Configure your ASP.NET project by choosing your desired project name, solution name, and location of project. Then click create at the bottom right.

7. You will now be prompted to select which type of ASP.NET Web application you want. Select the empty option and click create at the bottom right.

8. You now have the necessary prerequisites to start working with ASP.NET framework via VSC.
